Rust Information

When was Rust created and by whom?

Rust was created by Graydon Hoare in 2006 and offically released in 2010 while she was working with Modizilla Research.

General History

From when Rust started it has alwys been focused on saftey and performance. Rust has also always allowed different parts of the program to be executed out of order which allows parallel computing. This can improve the performace of an application. Rust's syntax is very simlair to C. The langauge is also very simple so low level programers can make small mistakes and not have their application crash on them.

    • Is Rust Still Being Used?

    Rust is currently the 14th most used programming language according to a poll taken in 2022. The language is still being used by huge tech companies such as Amazon, Microsoft, Google, and Meta. Companies like discord used the language to speed up there system and the company cloudfare uses it to process 20% of there internet traffic.
