C# Information

When was C# created and by whom?

The Language was created in 2000 by a Danish software devolper named Anders Hejlsberg who worked for Microsoft.

General History

Originally known as COOL which stood for "C-like object-oriented language" but changed due to copyright C# is one of the most popular coding languages as it is estimated about 28% of programmers use it. The language was originally intented to only be used by microsoft but evolved to be a very diverse programming language although you still mostly see it on microsoft. In it's early years C# and Java copied a lot of things off eachother until C# began going in a different direction. Overall C# has mostly evolved by upgrading the syntax from users overtime making the language easier to use.

What is C# still used to do?

  1. Advanced Business Applications
  2. Video Games
  3. Functional Web Applications
  4. Windows Applications
  5. MacOS
  6. Mobile Applictaions

Is C# still in use?

C# is one of the most popular programming languages today due to it's versatility and simplicity. The language also works well with the Unity Game Engine which is used to make a decent amount of games. Because of how simple the overall language is it is used to make apps like Micrsoft Visual Studio and Skype.
