When was PHP created and by whom?

Created by Ramus Lerdorf and first released in June of 1995.

General History

The language is a high level programming language that is dynamically typed. Originally used for handle form data on static websites. Since then it has been developed to become its own programming language used to interact with databases and servers. This is due to hortly after Lerford released PHP he then released source code for the language so devoplers could make their own edits.

What type of problems is PHP used for and is it still in use?

PHP nowadays is used for server side scripting. Companies like facebook still use it due to its many advantages such as receiving data from forms, generate dynamic page content, can work with databases, create sessions, send and receive cookies, and send emails. Overall the langauge is compatable with a lot of things and is a secure programming language.
