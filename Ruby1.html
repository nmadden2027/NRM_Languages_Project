Ruby Information

When was Ruby created and by whom?

Ruby was created by Yukihiro Matsumoto in 1993.

General History

Ruby was created because Matsumoto wanted an onject-oriented language that was also a easy to use scripting langauge. He added some features such as iterators, exception handling, and garbage collection on the original releace because he liked them. Overall Matsumoto wanted to create a simple language that programmers would love. Over time Matsumoto made the language stronger and it peaked in popularity in 2008.
